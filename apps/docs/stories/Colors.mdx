<Title>Colors</Title>

Brand color system generated from the brand hue. Every family shares the
same lightness/chroma curve — only the hue and peak chroma change — so the whole
palette stays visually consistent. Shade `500` is each family's brand step.

## Brand

`primary` is the brand teal; `secondary` and `tertiary` are cool harmonics;
`neutral` is a near-gray navy tint for surfaces and text.

<ColorPalette>{ramps(brand)}</ColorPalette>

## Status

Feedback colors for success, warning, error and info states.
